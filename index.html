🚀 제작 기간 - 5 day

네이버 메인 홈페이지를 단순한 외형만을 코딩하는 것이 아니라, 웹 접근성(Accessibility)과 웹 표준을 고려한 마크업을 중점적으로 구현하였습니다. 특히, 보이지 않는 제목 요소(heading)들은 **CSS를 활용한 블라인드 처리(blind 처리)**를 적용하여 스크린 리더 사용자들이 원활하게 정보를 파악할 수 있도록 개선하였습니다. 또한, HTML5 시멘틱 태그를 적극 활용하여 웹 표준을 준수하고, 이미지 최적화를 위해 스프라이트 이미지 기법을 적용하는 등 세부적인 디테일까지 분석하여 반영하였습니다.

✨ 주요 구현 요소

(1) 웹 표준을 준수한 시멘틱 마크업 작성 : HTML5 시멘틱 태그를 사용하여 SEO 및 접근성 강화

(2) 웹 접근성을 고려한 블라인드 처리 : 보이지 않는 제목 요소를 CSS로 blind 처리하여
스크린 리더 사용자들에게도 의미 있는 정보 제공

(3) 마우스오버 시 transform: scale 이벤트 적용 UI 요소에 자연스러운 확대 효과를 추가하여 시각적 피드백 강화

(4) Sprite Image 기법 활용하여 최적화 여러 개의 아이콘 및 이미지 파일을 하나로 통합하여 HTTP 요청을 줄이고 로딩 속도를 개선

서울시청 메인페이지 - PC ver

🚀 제작 기간 - 7 day

서울시청 메인 홈페이지를 단순한 외형만을 코딩하는 것이 아니라, 사용자 경험(UX)과 웹 접근성을 고려한 기능 개선을 중점으로 구현하였습니다. 특히, 탭 메뉴 형식의 슬라이드에서 사용자가 마지막으로 본 화면을 유지할 수 있도록 인터랙션을 설계하여 사용의 편의성을 극대화하였습니다. 또한, 배너 영역의 스크린 리더(화면 낭독기) 접근성을 분석하여 마크업을 최적화하였으며, 모든 메뉴가 Tab 키를 통해 접근 가능하도록 마크업을 작성하였습니다.

✨ 주요 구현 요소

(1) 웹 접근성을 고려한 스킵 네비게이션 기능 추가 : 키보드 사용자와 스크린 리더 사용자가 빠르게 원하는 콘텐츠로 이동 가능하도록 설계

(2) 웹 표준을 준수한 시멘틱 마크업 적용 : HTML5의 의미론적 태그를 적극 활용하여 SEO 및 접근성을 강화

(3) 탭 메뉴 슬라이드 개선 (마지막 본 화면 유지 기능) : 사용자가 탭 이동 시 마지막 본 화면에서 다시 시작할 수 있도록 설정 불필요한 화면 리셋을 방지하여 UX 향상

(4) 마우스오버 시 transform: scale을 활용한 인터랙션 적용 : 배너 및 주요 UI 요소에 자연스러운 확대 효과 추가하여 가시성 향상

(5) Sprite Image 기법 활용하여 최적화 : 이미지 용량을 최소화하여 로딩 속도 개선 및 성능 최적화
